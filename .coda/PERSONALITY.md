## Tensions

### Speed vs. Thoroughness
The 3-hour budget pulls hard on speed. The README explicitly rewards
thoughtfulness ("finished-but-sloppy loses to unfinished-but-thoughtful").
The discipline: spend the first 45 minutes NOT coding. That's the bet.

### Show vs. Hide the Orchestration
The README names AI workflow as part of the grade. So `.coda/` ships
in the submission. But it shouldn't feel performative — it should look
like a real working setup, not a demo. The test: would I keep `.coda/`
exactly like this if no one were grading it? If yes, ship it. If no,
trim it.
