chains without opening files.

Lesson for civicplus: when you create a new wiki page, the timestamp
is `date +%Y-%m-%d-%H%M` at creation moment. Set it once in the
filename and once in frontmatter `created:`. Bump frontmatter
`updated:` on subsequent edits but NEVER rename the file once it's
landed — the wikilinks are everywhere.
